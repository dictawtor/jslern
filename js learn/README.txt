🎯 روش‌های تبدیل به Word:
═══════════════════════════════════════════════════════════════════════════════

گزینه 1️⃣ : استفاده از فایل HTML (ساده‌ترین)
───────────────────────────────────────────────────
1. JAVASCRIPT_STUDY_GUIDE.html را باز کنید
2. Ctrl+A (انتخاب همه)
3. Ctrl+C (کپی)
4. Microsoft Word را باز کنید
5. Ctrl+V (چسب)
6. Save as "JAVASCRIPT_STUDY_GUIDE.docx"

گزینه 2️⃣ : استفاده از Google Docs (آنلاین)
───────────────────────────────────────────────────
1. docs.google.com بروید
2. سند جدید ایجاد کنید
3. محتوای Markdown یا HTML را پیست کنید
4. File → Download → Microsoft Word

گزینه 3️⃣ : استفاده از Pandoc (حرفه‌ای)
───────────────────────────────────────────────────
1. Pandoc را نصب کنید (choco install pandoc)
2. در Command Line اجرا کنید:
   pandoc JAVASCRIPT_STUDY_GUIDE.md -o JAVASCRIPT_STUDY_GUIDE.docx
